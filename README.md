# Site Casamento

API backend para um site de casamento, permitindo gerenciamento de convidados,
mensagens, presentes e galeria de fotos.

## Tecnologias
- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- Swagger / OpenAPI
- Maven

## Funcionalidades
- Cadastro e confirmação de convidados
- Envio de mensagens
- Lista de presentes com reserva pública
- Galeria de fotos
- Endpoints públicos e privados

## Como rodar localmente

1. Clone o repositório
2. Configure o profile local:
   SPRING_PROFILES_ACTIVE=local
3. Configure o banco MySQL local
4. Rode a aplicação

## Segurança
Credenciais sensíveis não estão versionadas.
O projeto utiliza variáveis de ambiente para configuração.
